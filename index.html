Getting Started

1

Include the Script

Add the aether-icons.js file to your project and include it in your HTML. Place it just before the closing </body> tag.

<script src="aether-icons.js"></script>
That's it — no build tools, no dependencies, no configuration needed.
2

Use an Icon

Place an <i> tag anywhere in your HTML with the data-aether-icon attribute set to the icon name.

<i data-aether-icon="heart"></i>
3

Style with CSS

Icons inherit currentColor and can be sized using standard CSS properties like width, height, and color.

<!-- Red heart, 32px -->
<i data-aether-icon="heart" style="width:32px; height:32px; color:tomato;"></i>
